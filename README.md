# 文件管理系统 python1298

本项目是一款基于Python的文件管理系统，采用Flask作为Web框架，搭配MySQL数据库以及HTML前端技术，实现了便捷的文件管理功能，并具备详细的日志记录功能。

## 技术栈

- 后端：Python 3.x，Flask
- 数据库：MySQL
- 前端：HTML，CSS，JavaScript

## 功能模块

- 文件上传与下载
- 文件分类管理
- 文件搜索
- 权限控制
- 日志记录

## 系统角色

- 系统管理员：负责系统的维护和用户权限管理
- 普通用户：可上传、下载、管理个人文件

## 运行环境

- 操作系统：兼容Windows、Linux、MacOS
- Python版本：3.x
- 需安装Flask、MySQL数据库及相关的Python库

## 部署步骤

1. 安装Python环境
2. 安装Flask及其他依赖库
3. 配置MySQL数据库
4. 部署前端文件
5. 运行Flask应用程序
